# ORGRAPHIC STUDIO

## Vision

Créer un assistant IA dédié aux artistes visuels.

Objectifs :

- Développer l'audience Instagram et Facebook.
- Générer automatiquement des publications.
- Analyser les statistiques.
- Gagner du temps.
- Augmenter les ventes.
- Construire une véritable marque artistique autour d'Orgraphic.
